## <a name="introduction">ğŸ¤– Introduction</a>

Bazario is a modern full-stack eCommerce application built to provide a smooth and intuitive online shopping experience. It offers users the ability to browse products, apply advanced filters, manage carts, and complete secure payments with ease. The platform is powered by a fast MERN backend, optimized database queries, and a clean, responsive UI built with React, Tailwind, and Shadcn components. With integrated Cloudinary for media handling and PayPal for checkout, Bazario ensures reliable performance, seamless navigation, and scalable architecture suitable for real-world commerce needs.

## <a name="quick-start">ğŸ¤¸ Quick Start</a>

Follow these steps to set up the project locally on your machine.

**Prerequisites**

Make sure you have the following installed on your machine:

- [Git](https://git-scm.com/)
- [Node.js](https://nodejs.org/en)
- [npm](https://www.npmjs.com/) (Node Package Manager)

**Cloning the Repository**

```bash
git clone https://github.com/Rishivenkatakousik/Ecommerce
cd Ecommerce
```

**Installation**

For Backend

```bash
cd server
npm install
```

For Frontend

```bash
cd client
npm install
```

**Set Up Environment Variables**

Create a new file named `.env.local` in the root of your project and add the following content:

```env
CLIENT_BASE_URL =
CLOUDINARY_API_KEY =
CLOUDINARY_API_SECRET =
CLOUDINARY_CLOUD_NAME =
MONGO_URL =
PAYPAL_CLIENT_ID =
PAYPAL_CLIENT_SECRET =
PORT =
```

**Start the development servers**

Run the backend

```bash
cd server
npm run dev
```

Run the frontend

```bash
cd client
npm run dev
```

Open [http://localhost:5173/](http://localhost:5173/) in your browser to view the project.
